# v0.7.0 Visual Enhancements Design

**Date:** 2026-07-23
**Version:** 0.7.0
**Status:** Approved

## Goal

Add 6 visual enhancements to the default template and theme system, making flexfetch output more visually distinctive and customizable.

## Features

### 1. Gradient Text

Apply color gradients to title and section headers using the theme's gradient color palette.

**Implementation:**

- `theme.rs`: Add `gradient_text(text, colors) -> String` function that applies per-character color cycling
- `template.rs`: Expose `theme_title_gradient` variable (title with gradient applied)
- Config: `display.gradient_title: bool` (default: true)

**Visual:**

```
╭─ System           ← gradient across "System" text
```

### 2. Progress Bars

Full progress bars for CPU and Memory (disk already has them), plus Battery.

**Implementation:**

- Template: Replace `(84%)` with `[████████░░] 84%` for memory, `[%]` for CPU
- Colors: Green (<60%), Yellow (60-85%), Red (>85%)
- Config: `display.progress_bars: bool` (default: true)

**Visual:**

```
│  CPU     : 12th Gen Intel... [██████░░░░] 54%
│  Memory  : 12.9 / 15.3 GiB  [████████░░] 84%
│  Battery : [████████░░] 80% (Not charging)
```

### 3. Box Styles

Multiple border styles for section headers and separators.

**Implementation:**

- Config: `display.box_style: "rounded" | "double" | "dotted" | "thick" | "ascii"`
- Predefined style constants in `template.rs`

**Styles:**

| Style   | Header | Row  | Separator |
| ------- | ------ | ---- | --------- |
| rounded | `╭─`   | `│`  | `────`    |
| double  | `╔═`   | `║`  | `════`    |
| dotted  | `┆─`   | `┆`  | `┄┄┄┄`    |
| thick   | `┣━`   | `┃`  | `━━━━`    |
| ascii   | `+-`   | `\|` | `----`    |

### 4. Logo Variants

Add 5 more distro logos + pixel-art mode for supported terminals.

**New logos:** Fedora, openSUSE, Alpine, Void, NixOS

**Pixel-art mode:**

- When `display.pixel_logo: true`, use Unicode block characters (▀▄█) to render a simplified pixel version of the logo
- Smaller footprint, works better in narrow terminals

**Implementation:**

- `logo.rs`: Add `pixel_logo(name) -> Option<Vec<String>>` with block-char art
- `image_logo.rs`: Respect `pixel_logo` config to switch rendering mode

### 5. Color Blocks

Show palette as colored blocks using ANSI 256-color or truecolor, not just `██`.

**Implementation:**

- Config: `display.palette_style: "blocks" | "squares" | "dots"`
- `template.rs`: Generate colored blocks with background colors

**Visual:**

```
│  ██ ██ ██ ██ ██ ██ ██ ██   ← blocks (current)
│  ▪ ▪ ▪ ▪ ▪ ▪ ▪ ▪ ▪ ▪ ▪   ← dots
│  ■ ■ ■ ■ ■ ■ ■ ■ ■ ■ ■   ← squares (Unicode)
```

### 6. ASCII Art Borders

Decorative frame around the entire output using ASCII art characters.

**Implementation:**

- Config: `display.frame: "none" | "single" | "double" | "decorative"`
- `template.rs`: Wrap output in frame characters
- Frame uses theme section color

**Visual:**

```
╔══════════════════════════════════╗
║  ┌ Arch Linux ASCII logo...     ║
║  │                              ║
║  │ ╭─ System                    ║
║  │ │  OS      : CachyOS         ║
║  │ ...                          ║
╚══════════════════════════════════╝
```

## Config Changes

```toml
[display]
gradient_title = true
progress_bars = true
box_style = "rounded"     # rounded | double | dotted | thick | ascii
pixel_logo = false
palette_style = "blocks"  # blocks | squares | dots
frame = "none"            # none | single | double | decorative
```

## Affected Files

- `flexfetch-core/src/theme.rs` — gradient_text() function
- `flexfetch-core/src/config.rs` — new display options
- `flexfetch-core/src/template.rs` — box styles, progress bars, frame, palette rendering
- `flexfetch-core/src/logo.rs` — 5 new distro logos + pixel-art
- `flexfetch-cli/src/main.rs` — pass new config to template
- `templates/default.tera` — conditional rendering for new features
- `docs/superpowers/specs/2026-07-23-v070-design.md` — this spec

## Verification

1. `cargo build --release` — no errors
2. `flexfetch` — default output with all features enabled
3. `flexfetch --box-style double --frame double` — double-line borders
4. `flexfetch --pixel-logo` — pixel-art mode
5. `flexfetch --palette-style dots` — dot palette
6. `flexfetch --no-gradient` — title without gradient
7. `flexfetch --no-progress` — text-only percentages
