Select all that interest you — we'll design each one

1

Side-by-Side Logo Layout

ASCII distro logo rendered next to info text (like neofetch). Currently logos exist but aren't shown in the default template.

2

Bar Graphs for Resources

Visual progress bars for CPU, Memory, Disk, Battery with color-coded usage levels (green → yellow → red).

3

Fix Kitty Protocol + Sixel

Kitty uses wrong protocol (currently uses iTerm2 escape). Sixel is stubbed out. Fix both for proper image rendering.

4

Gradient Title in Template

Wire the existing gradient_text() feature into the template so title/section headers can render RGB gradients.

5

Interactive TUI Mode

Live theme/module picker with real-time preview. Arrow keys to browse, enter to select. Uses ratatui.

6

New Theme Presets

Add more popular themes: Tokyo Night, Gruvbox Light, Rosé Pine, Everforest, Oxocarbon, etc.

7

Custom Section Ordering

Let users reorder sections (System, Software, Hardware, Palette) and toggle visibility per section.

8

Export Formats

Export output as SVG, HTML, or PNG (using image crate). Useful for README badges or screenshots.
