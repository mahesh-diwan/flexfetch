Feature 7: Custom Section Ordering

flexfetch.toml — section order
# config.toml
[display]
section_order = ["hardware", "software", "system", "palette"]
hide_sections = ["palette"]
separator = ": "
key_width = 10
# Result: Hardware first, no palette

Feature 8: Export Formats

CLI export flags
flexfetch --export svg -o output.svg
flexfetch --export html -o output.html
flexfetch --export png -o screenshot.png
flexfetch --export json -o data.json
# SVG = terminal-style vector graphic
# HTML = styled terminal card
# PNG = screenshot (via image crate)
# JSON = structured data (existing --format json)
